rentfax-api
===========

## installation
```
$ git clone git@github.com:codeforamerica/rentfax-api.git && cd rentfax-api
$ npm install
```

## running
```
$ npm start
```

## framework
`rentfax-api` is build using the [mach framework](https://www.npmjs.org/package/mach)
